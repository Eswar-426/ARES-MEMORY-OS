# Changelog

## [v0.20.0] - 2026-06-14

### Added
- **Project Memory**: Global project memory context engine.
- **Context Generator**: Intelligent summarization and context chunking for LLMs.
- **Retrieval**: Semantic vector-based episode retrieval with hybrid ranking.
- **Chat Import**: Automatic history imports from popular IDE tools.
- **Model Testing**: Comprehensive framework for assessing model performance metrics.
- **Dynamic Fallback**: Seamless multi-provider (NVIDIA, OpenRouter, Groq, Gemini) fallback chains.
- **Telemetry**: Detached SQLite persistence for agent evaluation metrics.
- **Dashboard**: Real-time React-based evaluation and provider health dashboard.

### Fixed
- Stabilized `ares-model-testing` build warnings.
- Resolved recursive CI/CD syntax checks.
- Cleaned up `.gitignore` hygiene for release artifacts.
